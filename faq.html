What is drip feed?

Drip Feed is a service that we are offering so you would be able to put the same order multiple times automatically.
Example:
Quantity:100
Runs: 10
Interval: 30


This will send 100 likes on your post every 30 minutes, in total, you will receive 1000 likes.
Never order more quantity than the maximum which is written on the service name (Quantity x Runs), Example if the service's max is 4000, you don’t put Quantity: 500 and Run: 10, because the total quantity will be 500x10 = 5000 which is bigger than the service max (4000). Also never put the Interval below the actual start time (some services need 60 minutes to start, don’t put Interval less than the service start time or it will cause a fail in your order).


MINIMUM INTERVAL SHOULD BE AT LEAST 4 TIMES THE START TIME.
We will not refund any orders that have not followed this rule.

How do I place an order?

Category: Select the social media platform or account you want to boost
Service: Select the service type
Description: This includes all details, quality, start time, speed per day, minimum and maximum quantity you can order, refill warranty, and price per 1k
Details: This will indicate samples and other pertinent information about the service.
Link: Enter the username or page link (ensure you are entering the right link as changes may not be possible once the order is placed)
Quantity: Enter the amount needed(do not use commas or dots, for example, use 1000 instead of 1,000)
Dripfeed: click this option if you want to place the same order

What is a refill?

Some social media platforms perform updates every few weeks or months to remove inactive accounts, this might cause some numbers to drop, all you need to do if that happens is open up a ticket with your Order ID and we will refill if free of charge if it still falls during the refill period.

When am I not eligible for refills?

Some clients place orders with us after using previous providers, for example, they place an order of 1,000 followers for an account with tens or hundreds of thousands of followers, such accounts are bound to drop at some point due to the large numbers of previously bought followers, as such, we cannot offer refills for accounts that drop exceeds below the start count.
